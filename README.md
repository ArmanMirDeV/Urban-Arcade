## Project Name
**Urban Arcade** - An Engaging Online Game Library

## Purpose
Urban Arcade is a web application designed for gamers and indie game enthusiasts. It allows users to discover, explore, and support indie game developers by browsing detailed game information, viewing ratings, and installing games they like. The platform aims to create a community-driven environment for game discovery.

## Key Features
- **User Authentication:** Sign up, log in, and log out with email/password or Google Sign-In.  
- **Password Management:** Reset password functionality via email.  
- **Game Library:** Browse popular and trending games with ratings and descriptions.  
- **Game Details:** Click on a game to view detailed information, including cover images, ratings, and developer info.  
- **Responsive Design:** Works seamlessly on desktop, tablet, and mobile devices.  
- **Profile Management:** Users can view and update their profile information.  
- **Notifications:** Toast messages for sign-in success, errors, and actions.  
- **Smooth Animations:** Powered by Framer Motion for a polished user experience.  

## Technologies & NPM Packages Used
- **Frontend:** React (`react`), React Router (`react-router`)  
- **Authentication & Backend:** Firebase (`firebase`)  
- **Styling & UI:** Tailwind CSS (`tailwindcss`), DaisyUI (`daisyui`)  
- **Animations:** Framer Motion (`framer-motion`)  
- **Icons:** React Icons (`react-icons`)  
- **Notifications:** React Toastify (`react-toastify`)  
- **Build & Tooling:** Vite (`vite`, `@vitejs/plugin-react`)  
- **Linting & Type Support:** ESLint (`eslint`, `@eslint/js`, `eslint-plugin-react-hooks`, `eslint-plugin-react-refresh`), TypeScript types (`@types/react`, `@types/react-dom`)
